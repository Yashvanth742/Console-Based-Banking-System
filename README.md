# Console-Based Banking System (Java)

This project is a simple console-based banking system developed using Core Java.

## Features
- Show account balance
- Deposit money
- Withdraw money
- Menu-driven interface
- Input validation

## Technologies Used
- Java
- Core Java
- Scanner Class
- Loops and Switch Statements

## How to Run
1. Download or clone the repository
2. Compile the Java file
3. Run the program in any Java IDE or terminal
